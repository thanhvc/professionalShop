Herramientas. Correlación

Una cartera de inversiones debe estar compuesta por diferentes Activos que no se deprecien al mismo tiempo. Para mitigar este efecto se usan diferentes Activos no correlacionados como Bonos, Bolsa, Metales Preciosos, Divisas, Obras de Arte, etc. Cuando se opta por un determinado Activo, a su vez hay que diversificar, por ejemplo en Bonos comprando diferentes emisores y duración –maturity- o en Bolsa comprando compañías de diferentes países y sectores.

La correlación es un concepto usado para diversificar una Cartera, el rango de valores está entre (+100) y (-100). Sean A y B dos compañías con datos de correlación dentro de:

 

  • Intervalo [+80, +100]: Las rentabilidades de A y B evolucionan de forma similar. La compra de ambas no supone diversificación, sino lo contrario asociación, incrementando el riesgo.
  • Valor próximo a 0: No hay correlación entre A y B, puede diversificar.
  • Valor negativo: Las rentabilidades de A y B evolucionan de forma diferente, son compañías efectivas para diversificar.
  • Intervalo [-80, -100]: La evolución de rentabilidades de A y B es diametralmente opuesta. Cuando una sube la otra baja y viceversa.

 

Recuerde que ante movimientos bruscos del mercado, todas las acciones tenderán a comportarse de manera parecida, disminuyendo la efectividad de la correlación. Para mitigar el impacto de este escenario deberá usar otras técnicas de diversificación y Gestión del Dinero.
